# Bible
Read the Bible in the terminal with Python and Curses.

![Preview](preview.png)

## Installation

Using pipx:
```shell
$ python3 -m pipx install git+https://github.com/noahsc0tt/bible@master
```
Using pip:
```
$ python3 -m pip install git+https://github.com/noahsc0tt/bible@master
```

## Usage 
```shell
$ bible
```
Use the arrow keys to navigate and make selections.
